Análisis de Varianza (ANOVA)

¿Qué es ANOVA?

Concepto

El Análisis de Varianza (ANOVA) es una técnica estadística desarrollada por Ronald Fisher que permite comparar las medias de tres o más grupos para determinar si existen diferencias estadísticamente significativas entre ellos.

Comparación de μ₁, μ₂, ..., μₖ

Se utiliza cuando queremos probar si las diferencias entre las medias de varios grupos son mayores que las esperadas por el azar.

Hipótesis en ANOVA

Fundamentos

Hipótesis Nula (H₀)

μ₁ = μ₂ = ... = μₖ

Todas las medias poblacionales son iguales

Hipótesis Alternativa (H₁)

Al menos una μᵢ ≠ μⱼ

Al menos una media difiere significativamente

Supuestos del ANOVA

Requisitos

Normalidad

Los datos deben distribuirse normalmente en cada grupo

Homocedasticidad

Las varianzas deben ser iguales entre grupos

Independencia

Las observaciones deben ser independientes entre sí

Fórmulas Clave

Matemáticas

Suma de Cuadrados Total (SST)

SST = ΣΣ(Xij - X̄)2

Variabilidad total de los datos

Suma de Cuadrados Entre (SSB)

SSB = Σni(X̄i - X̄)2

Variabilidad entre grupos

Suma de Cuadrados Dentro (SSW)

SSW = ΣΣ(Xij - X̄i)2

Variabilidad dentro de grupos

Estadístico F

F = MSBMSW

Donde MSB = SSB/(k-1) y MSW = SSW/(N-k)

Proceso ANOVA

Metodología
1

Plantear hipótesis

Definir H₀ (no diferencia) y H₁ (existe diferencia)

2

Verificar supuestos

Normalidad, homocedasticidad e independencia

3

Calcular estadísticos

SSB, SSW, MSB, MSW y el estadístico F

4

Tomar decisión

Comparar F calculado con F crítico o usar valor p

5

Concluir

Interpretar resultados en contexto del problema

Aplicaciones Prácticas

Usos

Investigación

Comparar efectos de diferentes tratamientos en experimentos

Negocios

Evaluar diferencias en ventas entre sucursales

Educación

Comparar resultados de diferentes métodos de enseñanza
